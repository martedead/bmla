# BMLA

**BMLA** is a radius-consistent derivative-free solver for systems of nonlinear equations based on local affine models.

> **Project status:** Research preview. The public MATLAB implementation and reproducibility materials are being prepared from the frozen BMLA v4.1-RC research release.

## Intended use

BMLA is designed for black-box nonlinear systems

```math
F(x)=0,
```

especially when derivatives are unavailable or expensive and function evaluations dominate the computational cost.
